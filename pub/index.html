Introduction

Welcome! Easy Tree is a web javascript library that started off as a quick way for developers to add a more intuitive site navigation tool for users to use. It became apparent that this functionality could be broadened to quickly make tree diagrams of a variety of types. The getting starting page will provide a guide to what is required to use this library. The API page describes in more details how to use the library, and 3 examples of what can be done with this library are provided.
